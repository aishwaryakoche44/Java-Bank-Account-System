# Java Bank Account Management System

## Description
A simple console-based Bank Account Management System built using **Java**.  
Users can deposit money, withdraw money, and check their account balance through a menu-driven interface.  
This project demonstrates basic Java concepts such as **classes, objects, switch-case**, and **user input handling** with the Scanner class.

## Features
- Menu-driven console application
- Deposit money into the account
- Withdraw money from the account
- Check current balance
- Simple, interactive, and easy to understand

## How to Run
1. Clone or download the repository.
2. Open the project in a Java IDE (Eclipse, IntelliJ IDEA, VS Code, etc.).
3. Compile and run `BankProgram.java`.
4. Follow the on-screen menu to deposit, withdraw, or check balance.
